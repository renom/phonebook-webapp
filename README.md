## Task
Create a simple web app which implements a phone book storage interface and work with [the web service](https://github.com/renom/phonebook-webservice).

## Technologies
Nuxt.js, Bootstrap.

## Main page
A table with contants and phones. A contact may have more than one phone.
A create button and a search field are above the table.
Edit and delete buttons are in the column "Actions" within each row (nearby each contact).

## Create and edit forms
Fields to input a surname, name and patronymic.
A field to input a phone number, a button "Add" is nearby this field (this button is used to add extra phone numbers), a button "Remove" is nearby each extra field that was added via the "Add" button (this button is used to remove a phone number).
After a contact is created, display the contact in the contact table.
